# Saldos Prefeitura

Sistema para controle diário dos saldos das contas da prefeitura.

## Antes de publicar

1. No Supabase, abra SQL Editor.
2. Cole o conteúdo do arquivo `supabase.sql`.
3. Execute.
4. Na Vercel, adicione as variáveis de ambiente:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

Depois publique normalmente pela Vercel.
